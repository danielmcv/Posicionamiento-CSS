Valor static

En este valor de posicionamiento realmente no pasa nada simplemente le estas diciendo a tu elemento que se mantenga en una posicion estatica y encontraras este tipo de posicionamniento por default, asi que no es necesario poner este valor.

Valor relative

En este valor de posicionamiento el elemento esta posicionado relativamente en su posicion normal y cualquier contenido no sera ajustado para caber en el hueco que deja este elemento.

Valor absolute

En este valor de posicionamiento el elemento esta posicionado relativamente al ancestro posicionado mas cercano. Sin embargo, si un ningun ancestro esta posicionado este tomara el del documento y se movera con el SCROLL de la pagina. NOTA: Un "ancestro posicionado" es un elemento que tiene la propiedad "position:;" excepto "static".

Valor fixed

En este valor de posicionamiento el elemento esta posicionado relativamente al viewport, Esto quiere decir que siempre se mantendra en el mismo lugar, no importa si hacen SCROLL en la pantalla puedes utilizar las propiedades de direccion. Este tipo de elemento no dejara un hueco en el lugar que se posiciono por default.
